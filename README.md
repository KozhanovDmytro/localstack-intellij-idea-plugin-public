# LocalStack plugin for Intellij IDEA
## Overview
LocalStack Integrator is designed to streamline the process of managing and integrating your local AWS environment, via LocalStack, directly into your IntelliJ IDE.
Currently, the LocalStack Integrator offers support for AWS Simple Queue Service (SQS) and partial support for the AWS Simple Storage Service (S3). With its user-friendly interface, users can access various functionalities within the comfort of their IntelliJ environment. Additionally, it's possible to configure the LocalStack URL address, Region, and access and secret keys.
